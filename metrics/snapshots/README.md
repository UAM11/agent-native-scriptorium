# snapshots

这里用于放周期性的指标快照。

## 建议记录的最小指标

- Days Logged
- Deep Work Blocks
- Hours Est
- Tokens Est
- Papers Read
- Source Modules Read
- Interview Questions Practiced
- Resume Revisions
- Applications Sent
- Assets Promoted

## 命名建议

- 周度：`YYYY-Www.md`
- 月度：`YYYY-MM.md`

## 注意

- 这些指标允许是估计值
- 重点是趋势，而不是绝对精确
