# ideas

这个目录用于存放尚未正式实现、但值得持续追踪的各种想法。

什么时候用：

- 当一个想法还不适合写成 `solution` 或 `skill`
- 当它更像“未来值得做的东西”，而不是“现在已经验证的东西”
- 当你希望先保留问题意识、价值判断和初步实现路径

这里的想法不局限于产品：

- 产品方向
- 科研问题
- 创业构想
- 工作方法
- 生活实验
- 日常灵感
- 一时荒诞但可能长出价值的念头

推荐内容：

- 为什么这个想法值得做
- 它属于什么领域
- 它想解决什么问题，或者它为什么让你着迷
- 如果要继续推进，第一步可以怎么做
- 有哪些技术、产品、认知或现实层面的风险
